# 100x Marketer

**Your marketing team, in a terminal.**

We teach solo marketers, fractional CMOs, and founder-marketers how to run a 5-person team's output using Claude Code and a curated MCP stack.

## The Bet

Claude Code is exploding among developers. Adoption among marketers is near zero. We're the bridge.

We don't sell software. We sell a **setup**: a configured `~/.claude/` directory, a stack of MCPs wired to the tools marketers already use, and a library of skills that automate 60% of repeatable marketing work.

## The Stack We Teach

- **Claude Code** as the operator
- **CLAUDE.md** as the brand voice + role context
- **Skills** as repeatable jobs (SEO, reporting, outreach, content, ops)
- **Hooks** as guardrails (brand voice enforcement, no em dashes, etc.)
- **MCPs** as the tool layer (Notion, Gmail, Drive, Publer, GA4, HubSpot)

## Funnel

Short-form (TikTok, Reels, Shorts, X) → Free Skool → Paid Skool ($79/mo) → DFY install ($4,997+)

## Project Structure

- `brand/` — Voice, visual identity, naming, positioning
- `curriculum/` — The Marketing OS (12-module paid curriculum) and teaching methodology
- `skills/` — Source-of-truth skill library we ship to members
- `content/` — Short-form scripts, long-form scripts, newsletters, captions
- `assets/` — Logos, thumbnails, templates
