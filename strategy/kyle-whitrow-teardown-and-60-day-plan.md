## What Kyle's content actually does (visual + structural teardown)

Pulled from prior session's playwright analysis + public posts.

### Visual style (the spine we already documented)

- **Dark terminal aesthetic** as the visual brand — Claude Code window IS the visual identity
- **Cut cadence: 1.5-2.5 seconds per shot** on screen recordings
- **Cold open in 1.3 seconds** (matches Sabrina's 1.5s rule and the Hook-Substance-Payoff framework)
- **On-screen text upper-third**, large bold sans, white with drop shadow, snap-cuts in
- **Talking head is secondary** — usually picture-in-picture or A-roll cutaways. Screen recording is the hero.
- **Captions** word-by-word, white-on-black-stroke, lower-third
- **Hooks observed:**
  - "I built [thing] in [time]"
  - "Here's how I [outcome]"
  - "Stop using [tool], use [Claude Code thing] instead"
- Targets ~30s reel duration

### Content beats (what each video teaches)

Post types we observe him repeating:
1. **Skill teardown.** "Here's the skill that does [marketing job]." Show the prompt, show the output, link to skill.
2. **Replaced-this-with-that receipt.** "I replaced [SaaS] with this 30-line skill."
3. **Live build.** "I'm building [thing] in 2 minutes."
4. **Tool spotlight.** "This MCP changes how you [do thing]."
5. **Contrarian take.** "Stop doing [common thing]. Do [Claude Code thing] instead."

Frequency: roughly daily on IG Reels, with cross-posts to TikTok and Shorts. The compounding mechanism is **post density × hook quality × consistency**.

### Funnel (what he does after the view)

- **CTA in caption + bio link** — single funnel target, no clutter. The link goes to thevibemarketer.com.
- **Site is a sales page**, not a content portal. One offer, one price, one CTA.
- **DM-based support and Q&A** drives conversation and high-intent leads.
- **Affiliate-style influencer mentions** in the wider Skool community amplify (James's network sells Kyle's product).

### Why this funnel works (vs. random "buy my course" CTAs)

1. **The free community is the lead magnet AND the audience nurture container** — same place, no email-only dead zone.
2. **Skills are gifts, not gated content.** Every reel has a corresponding skill in the Library. Audience associates 100x Marketer with "the place where free skills live."
3. **Paid tier is a clear graduation**, not a paywall. Members who attend office hours and want more depth opt in.
4. **Newsletter is the only email thread** — single send per week, easy to read, no overlap with paid funnels.

## The big risks (and how to manage them)

1. **Burnout on daily posting.** Mitigation: batch 5-7 reels every Monday using our pipeline. The `video-pipeline` skill already exists and is shake-out-tested.
2. **Hook fatigue.** Mitigation: the brand-profile.md format competitor list (Sabrina, Kyle, mavgpt) has a deep well of hook patterns. Repurpose ruthlessly.
3. **Audience overlap with Kyle / James.** Mitigation: lean into "solo marketer at 10-200 person company" wedge. Kyle and James lean creator/agency. That seat is open.
4. **Free Skool community ghost town.** Mitigation: seed it with 10 friends + 30 days of pinned content + daily check-in posts before announcing publicly.
5. **No live demo of the product.** Mitigation: every reel IS a live demo. The product (skills) lives on the public GitHub repo. Distance from sale → use is zero.
